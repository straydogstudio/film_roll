Configuration

There are many ways to implement responsive images. Here I demonstrate the use of picturefill.js with FilmRoll to create a full screen carousel. Do the following:

  • See the full screen page for full screen images.
  • In each child, implement the picturefill span to specify your images (see picturefill.js for more information): 
    <div>
 <span data-picture data-alt="Three bears">
  <span data-src="small.jpg"></span>
  <span data-src="medium.jpg" data-media="(min-width: 400px)"></span>
  <span data-src="large.jpg" data-media="(min-width: 800px)"></span>
 </span>
</div>
  • When you instantiate the FilmRoll tell it not to resize: 
    var film_roll = new FilmRoll({
      container: '#film_roll',
      resize: false,
    });
  • Resize the carousel when picturefill has responded to resize: 
    .film_roll_pager {
      position: relative;
      top: -20px;
    }
