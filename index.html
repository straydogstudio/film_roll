Film Roll: jQuery Carousel

Film Roll is a lightweight carousel that focuses on one item, centering it in the view:

It works with both large and small items, can use external buttons, and more:

‹ Left Right ›

Installation / Configuration

  • Download jquery.film_roll.js (9kb) or jquery.film_roll.min.js (6kb)
  • Link it in your page: 
    <script src="js/jquery.film_roll.js"></script>
  • Add markup: 
    <div id="film_roll">
  <div>
    <a href="..."><img src="..."> </a>
  </div>
  <div>
    <a href="..."><img src="..."> </a>
  </div>
</div>
  • Fire it up: 
    <script>
fr = new FilmRoll({
  container: '#film_roll',
  pager: true
});
</script>
